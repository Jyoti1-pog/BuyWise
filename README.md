# BuyWise

🛍️ BuyWise

Shop smarter with AI that searches, compares & buys for you.

🚀 Overview

BuyWise is an AI-powered shopping assistant that lets users describe what they need in plain English. The system intelligently:

🔍 Searches for relevant products
⚖️ Compares specs and prices
🧠 Recommends the best options
🛒 Places orders (only after user confirmation)

✨ Features

Natural language product search
AI-powered recommendations (Top Pick, Runner-up, Best Value)
Side-by-side product comparison
One-click order placement
Order tracking dashboard
Chat-based modern UI

🧩 Example Flow

User: Best earbuds under ₹5000  
AI: Top Pick → Sony WF-C500 (₹3,990)  

User: Buy it  
AI: ✅ Order placed (5-day delivery)

🛠️ Tech Stack

Frontend: React / Next.js / Tailwind CSS
Backend: Node.js / Express
AI Integration: Gemini
